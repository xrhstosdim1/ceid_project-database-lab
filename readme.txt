ΟΔΗΓΙΕΣ ΓΙΑ GUI:

1. Αναπτύχθηκε σε netbeans οπότε θα ήταν και το προτιμότερο IDE για τον έλεγχο.
2. Στο .zip, υπάρχει φάκελος jars που περιέχει τα jars που χρειάζονται για την εκτέλεση.
3. Για να δουλέψει σωστά, κάνετε import το project όπως είναι στο .zip, στο IDE σας, και εισάγετε τα jars.
4. Αν ζητήσει main class για το connect class, δώστε την Welcome_DBA.
